Heading Element

Description

The heading is a very important part of HTML because search engines use the headings to index the structure and content of your web pages.

Syntax

Headings are defined with the h1 to h6 tags. h1 defines the most important heading. h6 defines the least important heading. Always put the tags inside < >

Example

<p>This is a heading</p>

Display

How element looks in the browser...

This is a heading

Paragraph Element

Description

Paragraphs are used to write things you want in the HTML page after the headings.

Syntax

start tag: <p>

content: paragraph text

end tag: </p>

Example

<p>The dog went for a long run</p>

Display

How the element looks in the browser...

The dog went for a long run

Link

Break Element

Description

The break element is used to insert a single line break. It has no end tag.

Syntax

start tag: < br >

content: paragraph text

end tag: none

Example

<p>The dog went for< br >a long run</p>

in the browser
